# BRANDING

Warianty logo dodane do projektu:

- `assets/branding/logo-text.svg`
- `assets/branding/logo-long.svg`
- `assets/branding/logo-grouped.svg`
- `assets/branding/logo-grouped-big.svg`
- `assets/branding/app-icon.svg`

Proponowane uzycie:

- Header aplikacji (gorny pasek): `logo-text.svg`
- Ekran startowy / puste stany / onboarding: `logo-grouped.svg`
- README / baner repo / prezentacja: `logo-long.svg`
- Duze grafiki promocyjne: `logo-grouped-big.svg`
- Ikona aplikacji launcher: `app-icon.svg` (do eksportu na PNG)

Uwagi techniczne:

- Expo `icon`, `adaptiveIcon` i `splash` wymagaja plikow PNG, nie SVG.
- SVG mozemy od razu uzyc w samym UI po dodaniu obslugi importu `.svg`.
